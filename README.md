# ДНД Клуб

Многостраничный сайт для ДНД клуба с двумя кампаниями.

## Структура

```
/                    - Главная страница
/dead-band           - The Dead Band
/dead-band/characters - Персонажи
/dead-band/characters/[id] - Карточка персонажа
/shards              - Shards of Night City
```

## Запуск

```bash
npm install
npm run dev
```

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (иконки)
